A small class for validating Australian Business Numbers (ABN)

= Installation

  gem install abn

= Usage

  require 'abn'

  ABN.new("12042168743").valid?
  => true

  ABN.valid?("12042168743")
  => true

  ABN.valid?("12042168744")
  => false
